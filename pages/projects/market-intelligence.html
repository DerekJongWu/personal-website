Overview

Built an AI-powered market intelligence application that analyzes market trends and discovers relevant startups using Claude AI with real-time web search capabilities. The application uses a two-stage AI research process: first identifying the top 5 emerging trends in any market, then discovering 5 real, relevant startups for each trend. The platform features a modern, responsive UI with secure server-side API architecture.

How It Works

1. User Query Submission

Users enter a market query (e.g., "AI Security", "Climate tech solutions") via the frontend form, which calls the `/api/analyze` POST endpoint.

2. Trend Analysis

First API call uses Claude AI with web search to identify the top 5 emerging trends or themes in the specified market, providing detailed descriptions of each trend.

3. Startup Discovery

For each identified trend, a second API call researches and finds 5 real, relevant startups, including company names, descriptions, and founding information.

4. Results Display

Frontend displays trends and startups in an organized, visual format with responsive grid layout, showing how each startup fits into the identified trends.

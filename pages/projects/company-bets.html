Overview

Built a real-time prediction platform where users can create and participate in yes/no bets on company-related questions. The platform features live balance updates, real-time bet tracking, an admin panel for resolving bets, and a comprehensive transaction system. Users start with $100 and can place predictions, change their bets, and receive automatic payouts when they win.

Business Logic

Betting Flow

  1. Users create bets with questions and closing times
  2. Other users place YES/NO predictions (costs $1 for first prediction)
  3. Users can change predictions before closing time at no extra cost
  4. Admin resolves bets after closing time
  5. Winners automatically receive $1 payout

Balance System

  • New users start with $100 balance
  • Each prediction costs $1 (deducted immediately)
  • Changing prediction is free
  • Winners get $1 back (net zero if they won)
  • Cancelled bets refund $1 to all participants
  • Users can request refills, admins approve
